EDA分析

EDA思路
1. 构建多维度宽表
Style、SKC、SKU 颗粒度；年月、销量、价格、GMV、利润、PV、Click、广告花费、促销等字段统一沉淀。
2. 多对象 EDA 分析
单变量分析 → YOY分析 → MOM分析 → Lag分析 → 发现相关性 → 统计多维影响系数范围 → 拆解 month / 分层系数范围。
3. 基础属性聚合分析
对象基础属性聚合 → 分群/聚类 → 发现相关性 → 统计聚类下普遍弹性系数 → 拆解 month / 分层系数范围。
